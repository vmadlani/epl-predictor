# EPL Score Predictor

A Premier League match score prediction model inspired by FiveThirtyEight's SPI methodology.

---

## First-Time Setup (do this once)

### 1. Install Python
Download Python from https://python.org/downloads/ — get version 3.10 or newer.
During installation on Windows, tick the box that says **"Add Python to PATH"**.

### 2. Download this project
Either:
- Download as a ZIP from GitHub and unzip it, OR
- If you have Git: `git clone <repo-url>`

### 3. Open a terminal in the project folder
- **Windows:** Right-click the folder → "Open in Terminal"  
- **Mac:** Right-click the folder → "New Terminal at Folder"

### 4. Install the required Python libraries
Paste this into the terminal and press Enter:
```
pip install pandas numpy scipy requests matplotlib jupyter
```
This installs everything the project needs. You only do this once.

---

## Running the Scripts

Run each script in order. Each one builds on the last.

```bash
# Download 4 seasons of Premier League data
python src/01_download_data.py

# (Coming soon) Clean and explore the data
python src/02_clean_data.py

# (Coming soon) Train team ratings
python src/03_train_ratings.py

# (Coming soon) Predict upcoming matches
python src/04_predict_matches.py
```
